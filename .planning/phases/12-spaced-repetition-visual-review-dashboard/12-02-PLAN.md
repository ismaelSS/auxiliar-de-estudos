```java
enum Priority { OVERDUE, TODAY, NONE }

Priority getThemePriority(Theme theme, StatsService statsService) {
    int dueCount = statsService.getDueCount(theme.getName());
    if (dueCount > 0) return Priority.OVERDUE;
    // Check if any question is due today
    ThemeStats ts = statsService.getThemeStats(theme.getName());
    if (ts != null) {
        long now = System.currentTimeMillis();
        long todayStart = LocalDate.now().atStartOfDay(ZoneId.systemDefault()).toInstant().toEpochMilli();
        long todayEnd = todayStart + 86_400_000L;
        for (QuestionScore qs : ts.getQuestions().values()) {
            long nr = qs.getNextReviewTimestamp();
            if (nr > todayStart && nr <= todayEnd) return Priority.TODAY;
        }
    }
    return Priority.NONE;
}
```

Sorting: Collect themes into 3 lists (OVERDUE, TODAY, NONE), sort each alphabetically by name, concatenate overdue + today + none.

Search: TextField textProperty listener filters the observable card list. When text changes, hide cards whose theme name doesn't contain the search text (case-insensitive). Use node.setVisible(false) + node.setManaged(false) for hidden cards.
