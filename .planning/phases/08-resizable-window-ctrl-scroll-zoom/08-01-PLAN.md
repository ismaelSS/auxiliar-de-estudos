  <name>Task 1: ScreenController — zoom state, event filter, and font size application</name>
  <files>src/main/java/org/IsmaelSS/controller/ScreenController.java</files>
  <action>
Enhance ScreenController with zoom infrastructure. The file is currently 28 lines with a simple HashMap-based screen registry.

**Add imports:** `javafx.scene.input.ScrollEvent`, `javafx.event.EventFilter`, `javafx.scene.Node` (or use fully qualified names inline).

**Add fields** (after the existing `screens` map):
- `private int baseFontSize = 14;` — current zoom font size, default 14px
- `private static final int MIN_FONT_SIZE = 10;` — lower zoom bound (per D-01: range ~10px to ~30px)
- `private static final int MAX_FONT_SIZE = 30;` — upper zoom bound (per D-01: range ~10px to ~30px)

**Modify `registerScreen(String name, Scene scene)`:**
After `screens.put(name, scene);`, add two calls:
1. `installZoomFilter(scene);` — attach Ctrl+scroll handler to this scene
2. `applyCurrentFontSize(scene);` — apply current zoom level to the new scene's root

**Add `private void installZoomFilter(Scene scene)` method:**
Use `scene.addEventFilter(ScrollEvent.SCROLL, ...)` — NOT `setOnScroll`. Per D-01, must use capture phase. The handler:
1. Check `event.isControlDown()` — if false, return (let normal scroll pass through)
2. If `event.getDeltaY() > 0`: increment `baseFontSize` by 1, capped at `MAX_FONT_SIZE`
3. If `event.getDeltaY() < 0`: decrement `baseFontSize` by 1, floored at `MIN_FONT_SIZE`
4. Call `applyCurrentFontSize(event.getScene())` — apply to the scene that received the event
5. Call `event.consume()` — critical for ReportsView ScrollPane coexistence (prevents double-scroll)

**Add `private void applyCurrentFontSize(Scene scene)` method:**
Per D-01: apply zoom via `root.setStyle("-fx-font-size: " + size + "px")`. Check `scene != null && scene.getRoot() != null` before applying. Set style on `scene.getRoot()`.

**Modify `switchTo(String name)`:**
After retrieving the scene and before `stage.setScene(scene)`, call `applyCurrentFontSize(scene)`. This ensures zoom persists across screen switches (ZOOM-03). Per D-02, ScreenController holds the font size state and applies zoom on screen switch.

**Add public getter** (for testing and future use): `public int getBaseFontSize() { return baseFontSize; }`

Do NOT change the constructor signature. Do NOT remove the existing `stage.setScene(scene)` and `stage.show()` calls in `switchTo`.
