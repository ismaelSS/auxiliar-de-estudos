    - `private StatsData load()` — per D-14, Check `new File(STATS_FILE).exists()`. If missing, return `new StatsData()`. Try-catch `mapper.readValue(file, StatsData.class)`. On IOException, log warning via `Logger.getLogger(getClass()).warning(...)` and return new StatsData() (per Common Pitfall 2 and 3).
    
    - `private void save()` — per D-03 (append-only/incremental, write after updates). Use `mapper.writerWithDefaultPrettyPrinter().writeValue(new File(STATS_FILE), data)`. Catch IOException log severe.
    
    - `public void recordRound(List<RoundResult> results)` — Per D-12 data flow: iterate results, call `recordSingleAnswer(result)` for each, call `recalculateOverall()`, call `save()`.
    
    - `private void recordSingleAnswer(RoundResult result)` — Per D-01 schema: get or create `ThemeStats` via `data.getThemes().computeIfAbsent(result.themeName(), k -> new ThemeStats())`. Increment themeStats.totalAnswered. If `result.wasCorrect()`, increment themeStats.totalCorrect. Get or create `QuestionStats` via `themeStats.getQuestions().computeIfAbsent(result.questionText(), k -> new QuestionStats())`. Increment qStats.answered. If `result.wasCorrect()`, increment qStats.correct.
    
    - `private void recalculateOverall()` — Per D-01: sum all themes' totalAnswered and totalCorrect, set on `data.getOverall()`.
    
    - `public String getHitRate(String themeName)` — Per D-04 + Common Pitfall 5: Get ThemeStats from data.getThemes().get(themeName). If null or totalAnswered == 0, return `"N/A"`. Else compute `(double) ts.getTotalCorrect() / ts.getTotalAnswered() * 100` and return `String.format("%.0f%%", rate)`.
    
    - `public String getOverallHitRate()` — Same pattern using data.getOverall(). Return "N/A" if totalAnswered == 0.
    
    - `public List<Map.Entry<String, Double>> getHighestErrorQuestions(int limit)` — Per D-04 + Pattern 3: Iterate all themes' questions. For each QuestionStats where answered > 0, compute `errorRate = 1.0 - ((double) qs.getCorrect() / qs.getAnswered())`. Collect into ArrayList of `Map.entry(questionKey, errorRate)`. Sort descending by value. Return subList(0, min(limit, size)).
    
    - `public ThemeStats getThemeStats(String themeName)` — Return `data.getThemes().get(themeName)` (nullable for no-stats themes)
    - `public OverallStats getOverallStats()` — Return `data.getOverall()`
    - `public List<String> getAllThemesWithData()` — Return new ArrayList<>(data.getThemes().keySet())
    
    Edge cases handled:
    - Missing stats file → start empty (Pitfall 2)
    - Corrupted JSON → log warning, start empty (Pitfall 3)
    - Zero answers for theme → "N/A" (Pitfall 5)
