# Plan 11-02: File + Question CRUD

## Objective
Create the Gerenciar tab content: file listing with create/delete, question editor with add/edit/delete and auto-save.

## Implementation

### ThemeLoader
- `saveTheme(String name, List<Question> questions)` → writes JSON via ObjectMapper to `themes/name.json`
- `deleteTheme(String name)` → deletes `themes/name.json`

### QuestionFileManagerView
- Top section: file list (`VBox` listing theme names), "+ Novo Arquivo" button, "[×]" delete button
- Bottom section (visible when a file is selected): scrollable list of question cards, each with:
  - Question text field
  - 5 option text fields (A–E)
  - Correct answer ComboBox (0–4)
  - Remove button per question
  - "+ Adicionar Questão" button at bottom
- No Scene — just a VBox returned as root

### QuestionFileManagerController
- Constructor: receives `ThemeLoader`, `ThemeSelectionController` (for refresh callback)
- `initialize()` → loads file list, populates top section
- "Novo Arquivo" → shows TextInputDialog for title → sanitizes: trim, lowercase, replace non-alphanum with `-`, append `.json` → `themeLoader.saveTheme(name, [])` → refresh list → call `themeSelectionController.refreshScores()`
- Delete → `themeLoader.deleteTheme(name)` → refresh
- Select file → `themeLoader.loadAllThemes()` → find theme by name → populate editor with questions
- Questions stored in `ObservableList<Question>` bound to editor
- Auto-save: on focus-lost for any text field or ComboBox change → `themeLoader.saveTheme(currentFileName, questions)`
- Add question → create new Question with next available ID → append → save
- Delete question → confirmation → remove → re-index IDs sequentially → save

### CSS classes to use
- `.background` — root VBox
- `.title` — section labels
- `.button-primary` — "+ Novo Arquivo", "+ Adicionar Questão"
- `.button-secondary` — delete buttons
- `.section-title` — section headers
- `.scroll-pane` — question list wrap
