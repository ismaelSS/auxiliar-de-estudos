**Purpose:** Deliver the core scoring engine — wrong answers cost -3, right answers award +2, scores are bounded, and the theme aproveitamento replaces the old hit rate. All stats are keyed by question ID for stability.

- Modified `StatsData.java` — `QuestionScore` inner class, `ThemeStats` uses `Map<String, QuestionScore>`, deleted `QuestionStats`
- Modified `StatsService.java` — migration, score-delta `recordRound()`, `getAproveitamento()`, `getLowestScoreQuestions()`, removed old methods
</objective>

               if (isOldFormat(data)) {

                   data = migrateOldFormat(data);

           }
       }

       private boolean isOldFormat(StatsData data) {
           for (StatsData.ThemeStats ts : data.getThemes().values()) {
               for (String key : ts.getQuestions().keySet()) {
                   // Old format keys = long question text sentences (len > 20 or contains space)
                   // New format keys = short IDs like "record-primary-purpose"
                   return key.length() > 20 || key.contains(" ");
               }
           }
           return false;
       }
       ```

       private StatsData migrateOldFormat(StatsData oldData) {
           StatsData newData = new StatsData();
           for (Map.Entry<String, StatsData.ThemeStats> themeEntry : oldData.getThemes().entrySet()) {
               StatsData.ThemeStats oldTs = themeEntry.getValue();
               StatsData.ThemeStats newTs = new StatsData.ThemeStats();
               newTs.setTotalAnswered(oldTs.getTotalAnswered());
               newTs.setTotalCorrect(oldTs.getTotalCorrect());
               // Per-question scores reset — old answered/correct not convertible to bounded score
               newTs.setQuestions(new HashMap<>());
               newData.getThemes().put(themeEntry.getKey(), newTs);
           }
           // Preserve overall aggregates
           newData.getOverall().setTotalAnswered(oldData.getOverall().getTotalAnswered());
           newData.getOverall().setTotalCorrect(oldData.getOverall().getTotalCorrect());
           return newData;
       }

       public String getAproveitamento(String themeName) {
           StatsData.ThemeStats ts = data.getThemes().get(themeName);
           if (ts == null || ts.getQuestions().isEmpty()) return "N/A";
           int weightSum = 0;
           for (StatsData.QuestionScore qs : ts.getQuestions().values()) {
               int score = qs.getScore();
               if (score < 0) weightSum += -3;
               else if (score > 0) weightSum += 2;
               // score == 0 adds 0 weight
           }
           return String.valueOf(weightSum);
       }
       ```

    7. **Add `getLowestScoreQuestions(int limit)`:**
       ```java
       public List<Map.Entry<String, Integer>> getLowestScoreQuestions(int limit) {
           List<Map.Entry<String, Integer>> entries = new ArrayList<>();
           for (Map.Entry<String, StatsData.ThemeStats> themeEntry : data.getThemes().entrySet()) {
               for (Map.Entry<String, StatsData.QuestionScore> questionEntry :
                       themeEntry.getValue().getQuestions().entrySet()) {
                   entries.add(Map.entry(
                           questionEntry.getKey(),             // question ID
                           questionEntry.getValue().getScore() // question score
                   ));
               }
           }
           entries.sort(Map.Entry.comparingByValue());  // ascending by score (lowest first)
           return entries.subList(0, Math.min(limit, entries.size()));
       }
       ```
